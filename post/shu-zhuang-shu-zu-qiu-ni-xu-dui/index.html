2020-07-01

从 111 到 nnn 枚举，考虑当前枚举到的位置 iii ，他对总的逆序对的个数的贡献就是他前面的权值大于他的权值的数的个数，所以我们在i的权值的位置修改树状数组，表示 iii 的权值这个数已经出现过了，那么到了后面就可以直接查询从 111 到 iii 的权值的位置的总和。

void add(int i,int t){
    while(i<=len){//这里的len是元素种类，具体见上一篇博客
        c[i]+=t;
        i+=lowbit(i);
    }
}

    int n;
    cin>>n;
    for(int i=1;i<=n;i++){

        cin>>t;
        add(t,1);

}
